## Sujet

Voici les diagrammes d’un programme de nombre magique avec une architecture MVC.

Le programme du nombre magique est un programme qui choisit un nombre aléatoirement entre 0 et 100, et demande à
l’utilisateur de le trouve. A chaque proposition le programme répond soit :

- Trouvé
- Trop grand
- Trop petit

L’utilisateur n’a le droit qu’à 10 propositions.

Voici un exemple de trace que vous devez obtenir une fois le programme terminé.

```
Il vous reste 10 tentatives.
Entrez une proposition de nombre :
50
Trop grand
Il vous reste 9 tentatives.
Entrez une proposition de nombre :
25
Trop petit
Il vous reste 8 tentatives.
Entrez une proposition de nombre :
32
Trop grand
Il vous reste 7 tentatives.
Entrez une proposition de nombre :
29
Trop petit
Il vous reste 6 tentatives.
Entrez une proposition de nombre :
30
Trouvé
```
